# P99 - Text into Image Steganography (AI Generated Overview)



P99 is a lightweight, command-line tool written in C++ that allows you to hide text messages inside `.png` images and extract them later. It utilizes the `CLI11` library for clean and robust command-line argument parsing.



## Features

*Encode Text: Hide confidential or arbitrary text strings directly within the pixel data of a PNG image.

 Decode Text: Extract previously hidden text from a modified PNG image.

 Command Line Interface: Uses `CLI11` for structured arguments and help menus.

 Windows Ready: Includes pre-configured build files for VS Code and an automated test batch script.



## Repository Structure

 `main.cpp` / `header.cpp` / `header.h`: Core application logic, steganographic algorithms, and CLI parsing.

 `test.bat`: Windows batch file to quickly test encoding and decoding workflows.

 `main.exe`: Pre-compiled executable for Windows environments.

 `.vscode/`: Configuration files for Visual Studio Code users.
